## 🚀 Featured Projects

### ⚖️ [patent-claim-analyzer](https://github.com/SMPBenita/patent-claim-analyzer)
An agentic **LangGraph** pipeline that analyses patent claims: decomposes
independent claims into elements, runs deterministic formality checks
(antecedent basis, terms of degree, dependency validity), and produces an
auditable report. Provider-agnostic LLM layer runs fully offline for testing.
`Python` · `LangGraph` · `pytest` · `CI`

### 🧪 [patent-eval-harness](https://github.com/SMPBenita/patent-eval-harness)
A reproducible **evaluation harness** for LLM outputs on patent/legal tasks.
Combines deterministic metrics with an LLM-as-judge, applies weighted scoring,
and diffs two runs to catch regressions — so evals objectively steer model
development.
`Python` · `LLM-as-judge` · `pytest` · `CI`
